返回主页
.pptjs中有文件开头和结尾,最好不要编辑.

中间部分如何编辑:

示例:
setPage(1,function(){
   el.add("div").sets({
     'height': '40%',
     'width': '40%',
     'position': 'absolute',
     'top': '30%',
     'left': '30%',
     'backgroundColor': 'springgreen',
     'z-index': '1'
   },{
    'innerHTML': ('该格式仍存在bug,敬请谅解!地址:'+location.href)
   });
});
1.setPage用来设置页面:
setPage(page-number,page-set)
2.bg,el分别用来设置背景和元素:
bg.color(color) 设置背景颜色
bg.image(image) 设置背景图片
el.add(tagName).sets(style-set,other-set) 添加元素

其余用法同Javascript
